How to play

Using an executable is not currently supported. You will need Python or use repl.it.


Assuming you have Python installed or are using Replit, do the following:

• Download/clone the repository.
• Do python run.py
• Done.

If you encounter any bugs, please join our Discord server (on the home page)
or make an Issue/Pull Request.
